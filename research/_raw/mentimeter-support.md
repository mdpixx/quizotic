## Pricing Plans (as of late 2025)

### Free
- $0/month
- Up to 50 participants per MONTH (not per session); resets on account creation date
- After 50 participant limit hit: 8-hour re-present window for same presentation; then 30-day lockout
- Unlimited presentations
- Unlimited slides (up to 100 per presentation)
- Almost all question types (Quick Form excluded)
- No slide imports
- No custom branding
- No integrations
- Data may be anonymized and used by Mentimeter for inspiration

### Basic
- $11.99/presenter/month (individual, USD)
- $8.99/presenter/month (education)
- Annual billing only
- Unlimited participants
- All question types including Quick Form
- Slide imports (.ppt, .pptx, .key, .pdf)
- Excel/CSV data export (incl. quiz results)
- Basic collaboration
- No custom branding

### Pro
- $24.99/presenter/month (individual, USD)
- $14.99/presenter/month (education)
- Annual billing only
- Everything in Basic
- Custom branding (logo, colors, fonts)
- Team templates
- Advanced collaboration
- Moderation tools
- Embed PowerPoint + Google Slides
- Private presentations

### Enterprise
- Custom pricing; minimum 10 licenses
- Typical range: $15,600–$31,400/year ($6.50–$13.08/user/month)
- Everything in Pro
- SSO (Single Sign-On)
- SCIM user group management
- User activity insights
- Dedicated support
- API access (implied)

## Key Help Articles Referenced

- How to create a Quiz Competition: Select Answer + Type Answer slides; scoring mechanics; leaderboard setup
- Participating in a Quiz Competition: participant view, scoring display
- How to participate in a presentation: menti.com + code / QR / link
- Share QR code: unique, persistent, camera-scannable
- Let audience connect via link: direct link sharing
- Import presentation file: .ppt, .pptx, .key, .pdf formats
- Embed Google Slides: Pro plan required
- Embed PowerPoint: Pro plan required
- PowerPoint Add-in: create Menti slides inside PowerPoint; theme auto-match
- Open Ended slides: up to 250 chars per response; moderation options
- Word Cloud: 25-char limit per word; remove input option
- 100 Points: up to 20 options; 150-char per item
- Quick Form: Pro-only; text + checkbox + date fields
- Scales: 1–10 range; spider map variant
- Q&A: unlimited submissions; upvote ranking
- Pin on Image: accessibility note (not usable by visually impaired)
- Who Will Win: votes hidden until reveal
- SCIM overview: Enterprise only
- Analyze results: AI grouping, trend view
- PDF and screenshots: export options post-session
- Asynchronous use: survey mode for self-paced; share link; embed in LMS
