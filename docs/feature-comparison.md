## Summary: What Quizotic Covers

### Full parity with Kahoot Free + Paid (core quiz):
- Live quiz with MCQ, T/F, multi-select, timer, points, leaderboard — all covered

### Full parity with Mentimeter Free (interactive):
- Polls, word cloud, open-ended, Q&A, rating, ranking — all covered in v1

### Features we ADD that neither has:
- WhatsApp join link
- INR / UPI / Razorpay payments
- GST invoice
- Streak bonus scoring
- Per-question + per-participant reports in free tier
- Low-bandwidth participant page (< 100KB)

### Features we skip (conscious choice):
- Mobile app (web-first)
- PowerPoint integration (Mentimeter niche)
- LMS integrations (v3)
- AI quiz generation (v2)
- Self-paced homework mode (v2)
- Team mode (v2)
