## Long description (≤1500 chars)

> Quizotic is a live interactive learning platform that combines the best of Kahoot (fast-paced quizzes, leaderboards) and Mentimeter (word clouds, polls, rankings, open-ended responses) — built for Indian schools, coaching institutes, colleges, and corporate trainers.
>
> Key features:
> • Live quiz sessions with real-time leaderboards
> • Interactive presentations with word clouds, polls, ratings, rankings, Q&A
> • AI quiz generation from documents, URLs, or topics (Gemini 2.5 powered)
> • AI-enhanced presentation mode — upload a .pptx, get suggested interactive slides
> • Multi-language quizzes (Hindi, English, regional Indian languages)
> • Ultra-lightweight participant page — loads under 100KB, works on 1-2 Mbps classroom WiFi
> • No app install — participants join via browser on any device
> • INR pricing via Razorpay (UPI, cards, netbanking)
> • PDF/CSV export of results
> • Free tier with 100 participants per session; Pro for unlimited
>
> Built for the Indian classroom reality — low bandwidth, mixed devices, rupee pricing, regional language support.
